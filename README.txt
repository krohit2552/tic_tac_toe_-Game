Tic Tac Toe Game

Description
This is a simple Tic Tac Toe game built using HTML, CSS, JavaScript, and Tailwind CSS for styling. The game allows two players to take turns and play Tic Tac Toe in a 3x3 grid. The game features a reset functionality, and congratulatory messages are displayed when a player wins. Additionally, sound effects are included for clicks and winning.

Features
Two-player mode: Allows two players to play against each other.
Winner Detection: Automatically detects and announces the winner.
Draw Detection: Announces a draw if all boxes are filled with no winner.
Reset Game: Allows players to reset the game at any point.
Sound Effects: Includes click sounds for moves and a congratulatory sound for winning.
Tailwind CSS Styling: The game is styled using Tailwind CSS for a modern and responsive design.
Technologies Used
HTML: For the structure of the game.
CSS: Basic styling and layout.
Tailwind CSS: For enhanced, responsive styling.
JavaScript: For game logic and interaction.
Audio: For sound effects.

Usage
How to Play
Open the game in your web browser.
Two players take turns clicking on the boxes.
The first player to align three of their marks (O or X) horizontally, vertically, or diagonally wins the game.
The game will announce the winner and display a congratulatory message.
Click the "New Game" button to start a new round or the "Reset Game" button to clear the board and reset the game.
Customizing the Game
Styling: Modify the style.css and Tailwind CSS classes in index.html to change the look and feel of the game.
Sound Effects: Change the sound files in the sounds directory to use different sound effects for clicks and winning.
